# Personal-Finance
Dashboard that show your finance info

This repository was converted to a small React app using Vite.

Run locally:

```bash
npm install
npm run dev
```

Open http://localhost:5173 (or the port shown by Vite).

Files of interest:
- [index.html](index.html) — app shell loaded by Vite
- [src/App.jsx](src/App.jsx) — main React component
- [src/main.jsx](src/main.jsx) — React entry
- [src/index.css](src/index.css) — styles
